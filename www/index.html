CuteExplorer

CuteExplorer is file manager mainly for N900/Maemo 5 but works also in desktop and Symbian platforms. It has basic file management methods (cut, copy, paste, rename, delete) and in Maemo 5 it can be used to send files via Bluetooth.

Future releases of CuteExplorer are going to get some extra functionality like opening file in custom application, search and modifying file attributes.

Version 1.1.1 can be found in fremantle extras using application manager or from here.
